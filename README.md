# SENTINELAI — MONITORAMENTO INTELIGENTE DE SEGURANÇA CIBERNÉTICA

## SOBRE O PROJETO

O SentinelAI é uma plataforma inteligente de monitoramento de incidentes cibernéticos desenvolvida para a Expotech.

O projeto integra Inteligência Artificial, Business Intelligence, Engenharia de Dados e Segurança Cibernética em uma única solução estratégica.

O objetivo do sistema é monitorar incidentes de TI, prever ameaças, identificar padrões suspeitos e auxiliar equipes de segurança na tomada de decisões através de dashboards interativos e análises inteligentes.

# DASHBOARD POWER BI

O dashboard SentinelAI responde perguntas estratégicas como:

- Quantos incidentes ocorreram?
- Quais ataques são mais frequentes?
- Qual é a severidade predominante?
- De quais países vêm os ataques?
- Qual é o impacto financeiro?
- Quais clientes foram mais afetados?
- Qual é a eficiência operacional da IA?
- Qual é o tempo médio de resolução?
- Qual é o prejuízo estimado?
- Como a IA auxilia no monitoramento?

# INTELIGÊNCIA ARTIFICIAL

A IA foi desenvolvida utilizando Machine Learning com DecisionTreeClassifier.

O sistema realiza:

- Classificação automática de severidade
- Identificação de ameaças críticas
- Simulação de respostas automáticas
- Análise de risco financeiro
- Monitoramento em tempo real

---

# BANCO DE DADOS

O projeto utiliza MySQL para armazenamento dos incidentes monitorados.

Principais tabelas:

- incidentes
- dataset_final

O banco armazena:

- Tipos de incidentes
- Severidade
- Origem
- Status
- País do ataque
- IP suspeito
- Prejuízo estimado
- Receita do cliente
- Crescimento financeiro
- Tempo de resolução
